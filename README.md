# Credent Asset Investment

Credent Asset Investment Website

## Description

Credent Asset Investment Website is a responsive web application designed for asset investment management. It is built using HTML, CSS, JavaScript (jQuery), and PHP for the backend form handling. The site features interactive charts, smooth scrolling, and dynamic content loading.

## Features

*   **Responsive Design**: Optimized for various screen sizes, from mobile devices to large desktops.
*   **Interactive Charts**: Uses Chart.js to display investment data dynamically.
*   **Contact Forms**: Functional contact forms with validation and AJAX submission.
*   **Smooth Scrolling**: Navigation links scroll smoothly to the corresponding sections.
*   **Sticky Header**: The navigation bar remains visible while scrolling.
*   **Custom Scrollbars**: Uses mCustomScrollbar for styled scrollbars.
*   **Swiper Carousel**: Implements Swiper.js for sliders.

## Getting Started

### Dependencies

*   Any modern web browser (Chrome, Firefox, Safari, Edge).
*   A web server (Apache, Nginx, or similar) with PHP support is required to process the contact forms.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/credent-asset-investment.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd credent-asset-investment
    ```
3.  Configure your web server to serve the project directory.

### Executing program

*   Open `index.html` in your browser to view the homepage.
*   To test the contact forms, ensure the project is running on a PHP-enabled server. The forms submit data to `form.php`.

## Project Structure

*   `index.html`: The main homepage.
*   `form.php`: PHP script handling form submissions.
*   `assets/`: Contains static assets.
    *   `css/`: Stylesheets.
    *   `js/`: JavaScript files.
        *   `main.js`: Main application logic and library initializations.
        *   `custom.js`: Custom scripts for charts and interactions.
        *   `chart.js`: Chart.js configuration and utilities.
    *   `img/`: Images.
    *   `fonts/`: Font files.
